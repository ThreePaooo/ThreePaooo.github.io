Node.js的安装和使用

发表于 2018-08-29

本篇博客是NodeJs的安装过程和使用方法.

什么是Node.js?

简单的说 Node.js 就是运行在服务端的 JavaScript。
Node.js 是一个基于Chrome JavaScript 运行时建立的一个平台。
Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行Javascript的速度非常快，性能非常好。

Node.js的安装

  1. 下载node.js
  • 32位安装包下载地址:https://nodejs.org/dist/v4.4.3/node-v4.4.3-x86.msi
  • 64位安装包下载地址:https://nodejs.org/dist/v4.4.3/node-v4.4.3-x64.msi
  1. 安装node.js
  • 打开安装包,然后一路点击next至安装完成.

Node.js的使用

以一个注册/登录的项目为例.

  1. 在VS code中创建一个文件夹RegistAndLogin,右键这个文件夹,点击在终端中打开.
  2. 在终端中输入npm init命令,出现需要输入的命令行,都可以回车跳过,使用默认的选项.完成之后会生成一个配置文件package.json,保存整个项目的配置信息.
  3. 在RegistAndLogin文件夹下创建一个public文件夹(名字可自定义),存放静态文件,在public文件夹内创建文件index.html(必须创建,且名字不可自定义).并且在RegistAndLogin文件夹下创建文件index.js.
