scrapy的安装和简单使用

发表于 2018-08-01

scrapy简介

Scrapy是一个为了爬取网站数据，提取结构性数据而编写的应用框架.可以应用在包括数据挖掘，信息处理或存储历史数据等一系列的程序中.
其最初是为了页面抓取(网络抓取)所设计的，也可以应用在获取API所返回的数据或者通用的网络爬虫.

scrapy安装

1.Anaconda

如果你使用的Python环境是Anaconda，那么一条命令就可以完成scrapy的安装.
conda install scrapy

2.python

如果你是用的是Python，那么你需要确保安装以下几个程序来确保scrapy的正常使用.

  1. Twisted
    下载地址:https://www.lfd.uci.edu/~gohlke/pythonlibs/#twisted
    选择适合自己的正确的版本进行下载

  2. pywin32
    下载地址:https://github.com/mhammond/pywin32/releases
    选择适合自己的正确的版本进行下载
  3. pip
    如果正在使用从Python.org下载的Python2>=2.7.9或者Python3>=3.4，则已经安装了pip，只需要对pip进行升级.
    python -m pip install -U pip

安装完以上几个程序之后，用pip来安装Scrapy.
pip install Scrapy

安装完成之后，在控制台输入Scrapy，检测是否安装成功.如果出现下面的画面，则Scrapy安装成功.

Scrapy简单使用

  1. 首先进入自己的工作目录

  2. 输入scrapy显示可用的方法

    • bench 运行benchmark测试
    • fetch 使用scrapy下载器下载给定的url，并将获取的内容送到标准输出
    • startproject 创建项目的时候使用
    • genspider 在项目中创建spider
    • setting 获取scrapy的设定，在项目运行时，该命令将会输出项目的设定值，否则会输出默认设定
    • shell 以给定的URL或者空启动scrapy shell，进入shell之后可以进行你想要的操作
    • version 输出scrapy的版本，配合-v使用，会同时输出python以及平台的信息，方便BUG提交
    • view 在浏览器中打开给定的url，并以scrapy spider获取到的形式展现
    • runspider 在未创建项目的情况下，运行一个编写在python文件中的spider

  3. 使用scrapy startproject创建项目

    • items.py 定义爬虫程序的数据模型，类似于实体类。
    • middlewares.py 爬虫中间件。
    • pipelines.py 管道文件,负责对spider返回数据的处理。
    • setting.py 负责对整个爬虫的配置
    • scrapy.cfg scrapy 基础配置
    • spiders目录 负责存放继承自scrapy的爬虫类。
