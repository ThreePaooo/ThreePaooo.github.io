MySQL的安装和简单使用

发表于 2018-08-08

MySQL 是最流行的关系型数据库管理系统，在WEB应用方面 MySQL 是最好的RDBMS(Relational Database Management System：关系数据库管理系统)应用软件之一.

MySQL的下载请访问官方网站:https://www.mysql.com

下面是MySQL的安装过程.

  1. 打开MySQL安装程序，第一个界面点击接受协议，进入第二个界面。选择默认选项，点击next.
  2. 点击next，会弹出一个窗口，点击窗口内的Yes.

  3. 点击excute，等待安装完成.
  4. 点击next到下面的页面，保持默认选项即可，点击next.
  5. 根据自己的需求设置密码的强度.
  6. 点击next至下面的界面，点击Execute，安装完成后点击next.
  7. 点击next至下面的界面，设置password，点击check，如果红框处的状态为绿色，点击next.
  8. 然后一路next就完成MySQL的安装了.

安装完成之后，在终端中输入pip install pymysql.

下面是MySQL的简单使用.
