Python常见错误

Posted on 2018-07-17 | In test

在python的基础知识学习中，由于基础的薄弱，导致写代码的过程中会出现各种各样的错误。但是初学者面对这些错误的时候，解决能力较弱。接下来列举几种常见的错误类型，并给出解决方案。

1.SyntaxError: ‘return’ outside function

语法错误：return在方法以外的地方使用。

解决方法：在方法体内使用return。

2.SyntaxError: invalid syntax

语法错误：语句不合规则

解决方法：看错误信息在第几行，就从第几行开始向上找错误。

3. ValueError: substring not found

值错误：子字符串未找到

解决方法：输入正确的值。

4.IndexError: list index out of range

索引错误：列表索引查出范围

解决方法：输入正确的索引。

5.AttributeError: ‘tuple’ object has no attribute ‘remove’

属性错误：元组对象没有属性 ‘remove’

解决方法：使用该对象拥有的，正确的属性。

6.IndentationError: expected an indented block

缩进错误：没有正确的缩进

解决方法：用tab键正常缩进。

7.TypeError: pop expected at least 1 arguments, got 0

类型错误:pop方法希望得到至少一个参数,但是现在参数为0

解决方法：调用的方法需要几个参数，就给几个参数。

8.KeyError: ‘fond’

键值错误：没有指定的键值’fond’

解决方法：输入正确的键值。

9.TypeError: must be str, not int

类型错误：必须是字符串类型，不能是整数类型

解决方法：进行字符串拼接的时候，所有变量使用字符串类型。
